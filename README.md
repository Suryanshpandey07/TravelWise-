# 🌍 TravelWise

TravelWise is a web application that helps users explore countries and view real-time weather conditions for better travel planning.


- Search for any country  
- View country details (capital, region, population, flag)  
- Check the live weather of the capital city  
- Save favourite destinations  
- Filter and sort results  
- Responsive design  

## 🔗 APIs Used

- REST Countries API  
- OpenWeather API  

## 🛠️ Technologies

- HTML  
- CSS  
- JavaScript (Fetch API)  

## 📂 Project Structure

TravelWise/
- index.html  
- style.css  
- script.js  
- README.md  

## ⚙️ How to Run

1. Clone the repo  
2. Add your OpenWeather API key in script.js  
3. Open index.html in the browser  

## 🎯 Purpose

This project demonstrates API integration, dynamic UI rendering, and use of JavaScript concepts like fetch and array methods.
